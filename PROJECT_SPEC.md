# Project: Suzag-Litecore (Math Problem Bank Solution)

## 1. 프로젝트 개요
- 목표: 학원장, 강사, 학생을 위한 맞춤형 수학 문제은행 및 학습 관리 시스템(LMS) 구축
- 핵심 가치: 강사의 잡무(문제 선별, 편집) 최소화 및 학생의 메타인지 강화

## 2. 주요 사용자 및 기능
### A. 강사/원장 (Tutor/Admin) - 'Litecore' 대시보드
1. **문제지 만들기**:
   - 단원/유형/난이도별 필터링 및 수동/자동 출제
   - AI 큐레이션: 보유 DB 기반 최적 문항 추천
   - HWP/PDF 업로드 시 OCR 및 AI 유형 자동 분류
2. **학생/반 관리**:
   - 학생별 학습 이행률, 성적 추이 모니터링
   - 4단계 채점 리포트 확인 (정답/부분정답/부분오답/오답)
3. **학습 처방 (AI 히트맵)**:
   - 취약 유형(Red Zone) 식별 후 유사/발전 문제(Twins) 자동 생성

### B. 학생 (Student)
- 온/오프라인 답안 제출 (OMR 또는 모바일)
- '오답 Zero' 프로세스: 틀린 문제는 맞출 때까지 변형 문제 풀이
- 본인의 강약점 히트맵 시각화

## 3. 핵심 비즈니스 로직 (등급별 기능)
- **Lite (진단형)**: 기본 문제 다운로드, 진단 테스트
- **Core (성장형)**: 학원 자체 DB 업로드(OCR), 자동 해설, 유형 분류
- **Pro (완성형)**: 학원 전체 경영 관리, 자체 교재 제작

## 4. UI/UX 가이드
- 대시보드: 카드형 레이아웃, 직관적 그래프(성취도/이행률)
- 시험지 편집: 한글(HWP) 호환성을 고려한 웹 에디터
